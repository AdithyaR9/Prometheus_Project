Unit Six Exercise

Write a program that will gather student names from the user until “Done” is entered. The program should then ask the user for the number of seats in the classroom.

When the number of seats is less than the number of students the program should display “Error not enough seats in your class”.

When the number of seats is greater than or equal to the number of students the program should assign each student a seat randomly.

Data Storage:

  • Names – ArrayList of Strings
  • Seats – ArrayList of Integers
